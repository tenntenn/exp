### [go-ast-ssa-visualizer](./go-ast-ssa-visualizer/)

Interactive web application for visualizing Go AST (Abstract Syntax Tree) and SSA (Static Single Assignment) form.

**Features:**
- AST/SSA visualization
- Bidirectional code synchronization
- Multi-file support (txtar format)
- Go Playground integration
- WASM-based (no backend server required)

**Tech Stack:** Go (WASM), React, TypeScript, Monaco Editor
